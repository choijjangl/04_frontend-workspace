라디오버튼과 체크박스 관련 input 태그의 타입

라디오버튼(type="radio)

- 제시한 여러개의 값들 중에서 오로지 한 개만 선택해야 될 때
(name 속성 값이 똑같은 것들 끼리 하나의 그룹으로 지어짐) - 제출시 현재 선택된 (checked) 라디오 버튼의 value 값이 넘어감 (해당 라디오 버튼 선택시 어떤 값을 넘길껀지 value값을 명시해줘야한다!)

성별 :

체크박스(type="checkbox")

-제시한 여러개의 값들 중에서 여러개의 선택가능할 경우 -제출시 현재 선택된(checked)체크박스의 value 값 넘어감 (넘기고자 하는 value값 명시)

select태그와 option 태그

- 사용자에게 직접 키보드로 값을 입력받는게 아니라 내가 제시한 여러개 옵션(목록)들 중에서 선택할 수 있게끔
- 제출시 현재 선택된(selected) option 값이 넘어감
- 각각의 option에 value 값을 명시하지 않을 경우 content(한국, 미국 등등...) 영역의 값이 넘어감.
- value 값을 명시했을 경우 value값이 넘어간다.

국적 :

HTML5에서 새로 추가된 기능 datalist

- 사용자에게 직접 키보드로 값을 입력받을 수도 있고, 목록에서 선택할 수도 있음
